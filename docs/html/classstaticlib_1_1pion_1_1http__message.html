enum staticlib::pion::http_message::data_status_type

Used to cache chunked data.

Defines message data integrity status codes

Enumerator:
STATUS_NONE 

No data received (i.e.

all lost)

STATUS_TRUNCATED 

One or more missing packets at the end.

STATUS_PARTIAL 

One or more missing packets but NOT at the end.

STATUS_OK 

No missing packets.
